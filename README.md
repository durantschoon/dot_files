# dot_files
My dotfiles except for emacs

## Clone this to ~/dot_files

Then run these in the shell

<pre>
ln -si ~/dot_files/.zshrc ~/
ln -si ~/dot_files/.aliases.zsh ~/
</pre>

then choose one of:

<pre>
ln -si ~/dot_files/.aliases_home.zsh ~/
</pre>

<pre>
ln -si ~/dot_files/.aliases_work.zsh ~/
</pre>
